# Introducción

En este Manual procederemos a realizar una breve explicación de cómo instalar y actualizar las terminales StandAlone V650p. La instalación debe ser realizada hacien uso de un cable USB a tipo C.

# Hardware necesario

- Terminal Verifone 650p
- Cable USB a tipo C
- USB pendrive

# Software necesario

- Verifone T650p Pre-requisites

# Descarga de Pre-requisitos y Aplicación

Descargue los archivos necesarios para la instalación desde [aquí](https://downloads.ationet.com). Los archivos a descargar son:

- Version de Termianl 650p
- Ota_Secure v3.45.1
- SetSponsor
- SDI

# Instalación

Primero deberan descargar los archivos previamente indicados en el pendirve para posteriormente poder cargarlos en la terminal.
Una vez realizado este paso, deberan abrir la parte posterior de la terminal donde se podra visualizar la bateria. 

IMAGEN 

Debajo de la misma se encuentra un pequeño boton para el cual deberan introducir un clip de papel u otro objeto de volumen similar.
Al mantener presionado este boton por 3 segundos abran ingresado al menu de operadores de la terminal desde el cual accederemos al menu de supervisor.

IMAGEN

Ahora deberan seleccionar la opcion de ***"Supervisor"*** para la cual se les solicitara una contraseña, la misma por default es **1668311**
