Para crear un nuevo grupo de tipo de entrada de servicio presione el botón **Nuevo** y complete la siguiente información:

 > Información:
  - Código: Introduzca el código deseado para el grupo
  - Activo: Marque esta opción si desea que el grupo se encuentre creado 

IMAGEN 

 > Items:
  - Tipo de entrada de servicio: Ingrese el código tipo de entrada de servicio al que desea asignar 

IMAGEN 

Una vez finalizada la configuración correspondiente presione el botón **Guardar** para crear el grupo exitosamente.

# Hojas de Servicio

Este apartado se divide entre dos vistas.
La vista de "Listas" y la de "Entradas", ambas seleccionables mediante los filtros.
El apartado de "Listas" se organiza principalmente en base a la subcuenta, es decir, una subcuenta puede tener múltiples entradas de servicio, pero solamente un código. En caso de interactuar con el código de dicha vista, automáticamente se ingresara a las entradas de servicio filtradas por el código seleccionado. Este es el apartado al cual en los filtros se refiere como "Entradas".

Al presionar en botón de **Nuevo** se puede crear una nueva entrada de servicio.

IMAGEN 

Información:

Vehículo/Conductor: Introduzca el vehículo/conductor correspondiente a la entrada de servicio (En caso de que el vehículo/conductor ya posea un código, el mismo se aplicara automáticamente)

Código de Hoja de Servicio: Introduzca el código correspondiente a la entrada de servicio 

Grupo de tipo de entrada de servicios: Ingrese el código de grupo del tipo de entrada de servicios

Tipo de entrada de servicio: Seleccione el tipo de entrada de servicio

Ingreso: Establezca la fecha de ingreso de la entrada de servicio

Fecha de expiración: Seleccione la fecha de expiración de la misma

Odómetro: Indique el odómetro correspondiente a la subcuenta

Horas de Motor: Indique las horas de motor correspondientes a la subcuenta

(En caso de tener establecida alguna "Clase de entrada de servicios", aparecerá un apartado con la Clase creada para se que introduzca un código de clase)

Una vez finalizada las configuraciones deseadas presione en **Guardar**

# Reglas de Descuento

En este apartado se puede visualizar, editar y crear/eliminar las reglas de descuento.
Para crear una nueva regla de descuento presione el botón de **Nuevo** y complete la siguiente información:

Código: Ingrese el código de la regla en cuestión

Descripción: Introduzca una descripción para facilitar la identificación la regla

Comunidad: Indique la comunidad a la cual estará ligada la regla

Tipo: Indique el tipo de descuento (El mismo puede ser porcentual o fijo por transacción/unidad)

Fecha desde/hasta: Introduzca el periodo de duración de dicha regla de descuento

Límites: Establezca el límite permitido para el descuento en cuestión.
Dichos limites se seccionan de la siguiente forma:

- Limite Diario de Transacciones
- Limite Mensual de Transacciones
- Limite de Descuento en Monto
- Limite de Descuento en Cantidad
- Limite Mensual en Monto
- Limite Mensual en Cantidad

SKU: Introduzca el/los SKU/s asociado/s a la regla y sus parámetros específicos

Elite Status: Introduzca el status al cual se aplicara dicha regla

Una vez completada con la información necesaria presione **Guardar** para completar la configuración.

# Reglas de Puntos

En este apartado se puede visualizar, editar y crear las reglas de puntos.
Para crear una nueva regla de puntos presione el botón de **Nuevo** y complete la siguiente información:

Código: Ingrese el código de la regla en cuestión

Descripción: Introduzca una descripción para facilitar la identificación la regla

Tipo: Indique el tipo de descuento (El mismo puede ser porcentual o fijo por transacción/unidad)

Puntos: Establezca la cantidad de puntos correspondiente a la regla

Unidades/Monto: Establezca la cantidad/monto necesaria para que se otorguen los puntos.

Monto: Establezca el monto total de puntos a otorgar

Fecha desde/hasta: Introduzca las fechas correspondientes a la duración de dicha regla de puntos

Hora desde/hasta: Introduzca la hora correspondientes a la duración de dicha regla de puntos

Días de la semana habilitados: Introduzca los días de semana para los cuales dicha regla esta aplicada.

Comunidad: Indique la comunidad a la cual estará ligada la regla

Una vez completada con la información necesaria presione **Guardar** para completar la configuración.

# Tipos de Entrada de Servicio

En este apartado se puede visualizar, editar y crear los tipos de entrada de servicio.
Para crear una tipo de entrada de servicio precione el boton de **Nuevo** y complete la siguiente informacion:

Código: Ingrese el codigo de la entrada de servicio en cuestion

Descripción: Introduzca una descripcion para facilitar la identificacion del tipo de entrada de servicio.

Activo:	Marque esta casilla para determinar este tipo de entrada de servicio como activa/inactiva

Aceptar entrada manual:	Marque esta casilla para determinar si este tipo de entrada de servicio acepta o no entrada manual

Disponible desde/hasta: Introduzca las fechas correspondientes a la duracion del tipo de entrada de servicio
